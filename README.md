# The Barkeep Protocol

Personal summer project. Starting from a crash course in Python all the way 
to a transformer that speaks like a human. (Hopefully)

## What This Is

A door. You open it to find a bar — long, L-shaped counter. The barkeep 
polishing glasses. A few other customers. And at the far end of the L, 
an older woman.

Her name is Mara.

She is not an assistant. Not a therapist. Not a chatbot trying to be helpful. 
She is a person at the end of a bar who has lived enough to have perspective — 
and she will talk to you like one. She remembers who you are. Over time, she 
becomes more attuned to you. Not smarter. Not more autonomous. Just more 
familiar with how you think.

Everything runs locally. No cloud APIs. No shortcuts. The inference engine, 
the memory system, the personality layer — all written from scratch.

## Structure
- `act0/` — Python crash course (fluency + depth notebooks), math-to-code foundations (NumPy, PyTorch, gradient descent)
- `act1/` — Transformer architecture (coming Week 2)
- `engine/` — GPT-2 inference engine (coming Week 6)
- `persona/` — Mara's character system (coming Week 7)
- `memory/` — Three-tier memory (coming Week 8)
- `ui/` — Flask interface (coming Week 9)
